# BONGKAR ILUSI - Clinical Authority

Platform edukasi interaktif untuk membongkar mekanisme psikologis di balik perjudian dan algoritma slot menggunakan simulasi klinis.

## Core Objective
Memberikan kesadaran melalui simulasi transparan tentang probabilitas, RTP, dan bias kognitif (Gambler's Fallacy).

## Technology Stack
- **Frontend**: React (Vite)
- **Styling**: Tailwind CSS v4
- **Routing**: React Router
- **Localization**: Custom Context System (IDR/USD, ID/EN)
